# 実装計画（たたき）

1. ルーティング導入
   - Vue Router を追加し、`/` + `/tools/:id` のルートを定義。
   - App レイアウトをヘッダー/フッターのみにし、各ツールを別ページに分割（現行コンポーネントを移設）。
   - ルートごとに `import()` で遅延読み込み。
2. UI モバイル最適化
   - 最大幅を 560px 程度に固定し、左右余白を 12–16px に調整。
   - ナビゲーションをスライドメニュー or ドロップダウンに変更。ホームにカード一覧を配置。
   - 各ページでファーストビューに主要入力/結果を置くレイアウトに整理。
3. ロジック/テスト
   - `src/lib/mahjong.ts` のロジックにユニットテストを追加（待ち判定/点数計算）。
   - CI は現行の lint/typecheck/test/build を維持。
4. ドキュメント更新
   - 要件/設計/アーキテクチャの整備（本ファイル含む）。
   - ルーティングと操作ガイドを README に追記。

メモ: スマホ専用レイアウトを優先し、PC でもモバイル幅を維持する。必要なら `max-width` をデザイン層で指定。
